## How to run
### Run spring boot directly
1. `./mvnw clean package`
2. `./mvnw spring-boot:run`

### Building snapshot
1. `./mvnw clean package`
2. `./mvnw package`
3. Go to target folder and file *.jar
4. Run `jar -jar <file>`
